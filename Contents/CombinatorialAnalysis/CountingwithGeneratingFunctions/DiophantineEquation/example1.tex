\begin{PROBLEM}
    \p
    تعداد جواب‌های معادله‌ی زیر را پیدا کنید.
        $$e_{1} + e_{2} + e_{3} = 17$$
        $$e_{i} \in W, 2 \leq e_{1} \leq 5, 3 \leq e_{2} \leq 6, 4 \leq e_{3} \leq 7 $$

    \SOLUTION[\REF{R} پاسخ]{
        \p
        تعداد جواب‌های معادله‌ی بالا با محدودیت‌های در نظر گرفته شده، با ضریب 
        $x^{17}$
        در بسط زیر برابر است (توجه کنید که هر پرانتز نشان‌دهنده‌ی مقادیر قابل اخذ توسط یکی از متغیر‌های صورت سوال هستند؛
        هر جمله‌ی داخل یک پرانتز، نشان‌دهنده‌ی یکی از مقادیر ممکن برای آن متغیر است):
        $$\underbrace{(x^{2} + x^{3} + x^{4} + x^{5})}_{e_1}\underbrace{(x^{3} + x^{4} + x^{5} + x^{6})}_{e_2}\underbrace{(x^{4} + x^{5} + x^{6} + x^{7})}_{e_3}$$
        $$= 1x^{9} + 3x^{10} + 6x^{11} + 10x^{12} + 12x^{13} + 12x^{14} + 10x^{15} + 6x^{16} + 3x^{17} + 1x^{18}$$
        از آنجایی که ضریب 
        $x^{17}$
        برابر است با تعداد دفعاتی که حاصل‌ضرب 
        $x^{e_{1}}$
        از عبارت اول و 
        $x^{e_{2}}$
        از عبارت دوم و 
        $x^{e_{3}}$
        از عبارت سوم
        به گونه‌ای که  
        $e_{1} + e_{2} + e_{3} = 17$
        باشد، پس با تعداد جواب‌های این معادله برابر است.
        \p
        بنابراین طبق نتیجه‌ی ضرب کردن بسط بالا، ضریب 
        $x^{17}$
        برابر ۳ می‌شود. بنابراین معادله دارای ۳ جواب است.
    }
\end{PROBLEM}
